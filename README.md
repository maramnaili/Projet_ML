#  Classification Autisme avec Machine Learning

##  Description
Ce projet utilise des techniques de Machine Learning pour prédire le diagnostic d'autisme (ASD vs Control) à partir de données de neuroimagerie et de tests cliniques.
Il s'appuie sur la base de données ABIDE (Autism Brain Imaging Data Exchange), une initiative internationale de partage de données d'imagerie cérébrale pour la recherche sur l'autisme.
Ce projet vise à appliquer des techniques de **Machine Learning** sur le dataset **ABIDE (Autism Brain Imaging Data Exchange)**.

## Objectif principal :
 Développer un modèle de classification capable de distinguer les individus avec un Trouble du Spectre de l'Autisme (ASD) des individus au développement typique (Control).

## Dataset
Base de données ABIDE
Taille : 1,112 sujets
Variables : 106 features
Classes :
1 = ASD (Autism Spectrum Disorder)
2 = Control (Développement typique)

Le travail comprend :
- le prétraitement des données
- l’analyse exploratoire
- la transformation des variables
- l’entraînement et l’évaluation de modèles de Machine Learning
